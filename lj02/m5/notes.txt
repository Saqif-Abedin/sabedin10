Aggregation Framework

    -more powerful than MQL
Syntax
db.listingsAndReviews.aggregate([
                                  { "$match": { "amenities": "Wifi" } },
                                  { "$project": { "price": 1,
                                                  "address": 1,
                                                  "_id": 0 }}]).pretty()

$group: takes data, siphons into distinct reservoir
{ "$group": { "_id": <expression>, //Group by expression
